五层协议

  • OSI 七层协议概念清楚、概念完整但是过于复杂且不实用；TCP/IP 协议在 OSI 协议完全出来时已经占领了市场，所以现在互联网都是基于使用 TCP/IP 协议；

  • OSI 七层协议：应用层=》表示层=》会话层=》运输层=》网络层=》数据链路层=》物理层

    TCP/IP 协议：应用层=》运输层=》网际层=》网络接口层

    因为 TCP 的最底层网络接口层实际没什么内容，所以教学一般使用结合了 OSI 和 TCP/IP 优点的五层协议

    五层协议：应用层=》运输层=》网络层=》数据链路层=》物理层

  • 五层协议的

    • 应用层：工作在具体应用进程之间的协议，通讯规则。比如 HTTP 协议，就是客户端（一台主机）浏览器（应用进程）和服务器（另一台主机）的应用服务（应用进程）的通讯协议。

      主要协议有：HTTP，DNS，SMTP 和 POP3、IMAP（邮件），DHCP（动态主机配置）

    • 运输层：工作一台主机的应用进程到另一台主机的应用进程之间，提供进程间的逻辑通道。比如 TCP 协议，规定主机间的通讯建立和断开，接收数据速度。

      主要协议有：TCP/UDP，

    • 主要协议：网际协议 IP、配套 IP 协议的 ARP、ICMP、IGMP

      ARP，Address Resolution Protocal，主要是根据 IP 得出对应的物理地址（MAC 地址）

    • 数据链路层

    • 物理层
